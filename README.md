# ❤️ LoveBirds — Cinematic Emotional Story Experience

A premium cinematic multilingual emotional storytelling website built with pure HTML, CSS, and JavaScript.

Designed to create an immersive emotional experience for couples, loved ones, and meaningful connections through dynamic storytelling, aesthetic visuals, hidden interactions, and emotional transitions.

---

# ✨ Features

- 🌙 Dynamic emotional quotes
- ❤️ Multilingual experiences
- ✨ Random story generation
- 🎆 Double-tap fireworks system
- 🔐 Secret password unlock
- 🧠 Hidden clue interactions
- ⌨️ Cinematic typewriter animation
- 💫 Floating hearts animation
- 🎭 Emotional paragraph endings
- 📱 Fully responsive mobile UI
- 🌌 Random background system
- 🎨 Glassmorphism premium interface

---

# 🌍 Supported Languages

- English
- हिंदी (Hindi)
- मराठी (Marathi)

Each language delivers a completely unique emotional storytelling experience instead of direct translations.

---

# 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- hosted on vercel

No frameworks.
No libraries.
Pure frontend creative engineering.
